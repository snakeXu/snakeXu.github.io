(1)npm config set registry http://registry.npm.taobao.org/
(2)npm install
(3)进入文件夹，找到shells/chrome/build/unpacked文件夹
(4)打开chrome浏览器扩展，将步骤3的文件夹移到扩展页面
至此react-devtools安装完成，打开本地react项目的浏览器开发工具可以看到在console/network···后有了react，如图
