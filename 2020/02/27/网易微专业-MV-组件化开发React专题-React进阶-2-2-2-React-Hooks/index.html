网易微专业-MV*&组件化开发React专题-React进阶-2.2.2-React Hooks

ReactHooks就是用函数式组件把class组件的方方面面实现的过程。组件包括(1)props、state，也就是组件的数据；(2)组件的生命周期
1.useState&useEffect
简单例子:
1-1.了解如何定义数据
index.js

import React from 'react';
import ReactDOM from 'react-dom';
import App_10 from './App_10';
ReactDOM.render(
  ,
  document.getElementById('root')
);
App_10.js 
import React,{useState,useEffect} from "react";
const App_10 = (props)=>{
    //下面两个变量相当于定义state，
    //在这个定义过程中不允许if{}else{}的条件式定义，也就是每次执行的时候定义的变量数量不能变
    let [time,setTime] = useState(0);//useState传入的是初始值，setTime是修改time事件
    let [age,setAge] = useState(10);
    return (
        < div>
            Hello World {props.name}< br/>
            {time}
            < button onClick={(e)=>{setTime(time+1)}}>add< /button>< br/>
            {age}
            < button onClick={(e)=>{setAge(age-1)}}>cut< /button>
        < /div>
    )
}
export default App_10



1-2.了解如何定义生命周期

import React,{useState,useEffect} from "react";
const App_10 = (props)=>{
    //下面两个变量相当于定义state，
    //在这个定义过程中不允许if{}else{}的条件式定义，也就是每次执行的时候定义的变量数量不能变
    let [time,setTime] = useState(0);//useState传入的是初始值，setTime是修改time事件
    let [age,setAge] = useState(10);
    //useEffect相当于componentDidMount和componentcomponentDidUpdate和在一起的作用
    useEffect(()=>{
        console.log('Hello')
    })
    return (
        < div>
            Hello World {props.name}< br/>
            {time}
            < button onClick={(e)=>{setTime(time+1)}}>add< /button>< br/>
            {age}
            < button onClick={(e)=>{setAge(age-1)}}>cut< /button>
        < /div>
    )
}
export default App_10


当我们直接写入useEffect的时候我们发现，只要页面数据有变化useEffect函数就会执行

因此我们给useEffect加入参数

useEffect(()=>{
    console.log('Hello')
},[])//对，你没看错，就是加入了一个为空数组的参数




useEffect(()=>{
    console.log('Hello')
},[age])


如果我们给useEffect添加一个参数为我们之前定义的age，那么会发现，只有在age变化的时候useEffect才会执行，如果time变化useEffect没有执行

useEffect(()=>{
    console.log('Hello')
    return ()=>{
        console.log('status clear')
    }
},[age])


老师说useEffect函数的return相当于componentWillUnMount，但是我发现还是在每次age变化的时候return里的函数就执行里，具体是怎么理解，我看接下来老师怎么讲吧
1-3.todolist

const Action = (props)=>{
    const [value,setValue] = useState('js')
    return (
        < div>
            < input value={value} onChange={(e)=>{setValue(e.target.value)}}/>
            < button onClick={(e)=>{props.onAdd(value)}}>add< /button>
        < /div>
    )
}
const List = (props)=>{
    return (
        < div>
            {props.data.map(item=>(
                < div key={item.id}>< span>{item.name}< /span>< button>del< /button>< /div>
            ))}
        < /div>
    )
}
const App_10 = (props)=>{
    const [data,setData] = useState([
        {name:'react',id:1},
        {name:'vue',id:2},
    ])
    return (
        < div>
            < Action onAdd = {(name)=>{
                data.push({name:name,id:Math.random()})
                // setData(data)这样用不可以，js判断一个对象有没有变化的唯一依据和标准是他的内存地址有没有发生变化，
                //push的时候改变的只是数据的值，并没有改变内存地址，所以js当作这个数据没有发生变化
                setData([...data])
            }}/>
            < List data={data}/>
        < /div>
    )
}
export default App_10

3.自定义hook
自定义hook的主要用途就是实现组件的公共功能

const useTitleHook = (title)=>{
    useEffect(()=>{
        document.title = title
        return ()=>{
            document.title = 'title'
        }
    })
}
const App_10 = (props)=>{
    let [title,setTitle] = useState('hello');
    useTitleHook(title);
    return (
        < div>
            < input value={title} onChange={(e)=>{setTitle(e.target.value)}}/>
        < /div>
    )
}
export default App_10
