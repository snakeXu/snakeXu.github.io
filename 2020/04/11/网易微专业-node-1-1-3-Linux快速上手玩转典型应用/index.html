2.常用命令
2-1.查看ip
ifconfig mini版没有安装此命令，需要自己安装
ip addr 显示ip信息
2-2.yum命令
·yum是一个在CentOS中shell前端软件包管理器
·yum update 更新所有的软件
·yum install 安装指定的软件
·yum remove 删除指定的软件
···
yum install net-tools可以安装上面提到过的ifconfig(ifconfig包含在net-tools中)。但是执行安装程序需要连网，在虚拟机里就是要设置网络和IP等，需要设置一个文件，再在虚拟机选项中选择桥接模式，再重启虚拟机。
2-3.vi/vim命令
·命令模式
·i 切换到输入模式
·x 删除当前光标所在处字符
·: 切换到底线命令模式
·输入模式
·底线命令模式
·q 推出程序
·w 保存文件
·set number 显示行号
2-4.系统信息
·free -m 内存信息
·df -h 硬盘信息
·cat/proc/cpuinfo cpu信息
···
2-5.文件系统
·cd 进入目录
·ls 列出目录信息，ls-al(ll)列出详细信息
·touch 新建文件
·mkdir 新建目录
·rm 删除文件或目录
·cp 复制
·mv 移动(重命名)
···
2-6.搜索、查找、读取
·tail 从文件尾部读取
·head 从文件头部读取
·cat 读取整个文件
·more/less 分页读取
·grep 搜索关键字
·find 查找文件
·| 通过管道传递命令
···
2-7.压缩和解压
·tar 用于压缩和解压
·tar -czvf 打包压缩
·tar -tzvf 列出压缩文件内容
·tar -xzvf 解压文件
···
2-8.用户操作
·useradd/adduser 添加用户
·passwd 设置密码
·userdel 删除用户
·sudo 提权
2-9.更换yum源

这里需要通过wget将cent-os下载到本地，如果没有安装wget需要先安装wget，那么上面图中第一步的备份就先不要做，要先安装wget

3.玩转典型应用
3-1.防火墙
3-1-1.服务设置
(1)安装 yum install firewalld
(2)启动 service firewalld start
(3)检查服务状态 service firewalld status
(4)关闭/禁用 service firewalld stop/disabled
3-1-2.防火墙设置
·查询服务 firewall-cmd –list-services
·查询开放端口 firewall-cmd –list-ports
·查询端口 firewall-cmd –quert-port=80/tcp
·添加开放端口 firewall-cmd –add-port=3306/tcp
3-2.Apache
3-2-1.服务设置
·安装 yum install httpd
·启动 service httpd start
·检查服务状态 service httpd status
·关闭/重启 service httpd stop/restart
ps -ef 可以查看正在运行的进程
ps -ef | grep httpd 查看http进程
3-2-2.Apache设置
·虚拟主机设置
·伪静态操作
cd /etc/httpd
cd conf
vim httpd.conf
可以另外开一个窗口，看一下http服务相关的
cd /etc/httpd/
ll
可以看到列表中conf.modules.d是所有模块
这里还教了如何配置虚拟主机，也就是一个服务器上可以放置多个网站
伪静态就是类似于我们看到的博客详细页面是.html结尾的，但是这只是在服务器上设置的，并非是真的每篇文章一个静态.html文件，这个设置需要在服务器上设置，本节课也有讲到
